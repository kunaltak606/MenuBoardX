
# MenuBoardX - Digital Menu & Ordering Interface

MenuBoardX is a modern, mobile-first digital menu and cart system for restaurants and cafes. It features item browsing, live search, category filters, item add/edit, and a sticky cart footer.

***

## Features

- Responsive UI built with React
- Live search and category filter for menu items
- Add items with options to cart
- Edit items in cart with real-time update
- Sticky footer with current cart total and easy checkout access
- Clear cart functionality

***

## Getting Started

### Prerequisites

- Node.js (v14 or later recommended)
- npm or yarn package manager

### Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/menuboardx.git
cd menuboardx
```

2. Install dependencies:

```bash
npm install
# or
yarn install
```

### Running Locally

Start the development server:

```bash
npm start
# or
yarn start
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

***

## Project Structure

- `src/components/` – React components like Header, MenuMain, ItemModal, OrderView, FooterBar
- `src/App.js` – Root component managing app state (cart, modal, order view)
- `src/styles/` – CSS files for styling components
- `public/` – Static assets

***

## Usage

- Browse menu items by category or search
- Click "Add" on any item to open options modal
- Add configured item to cart
- View and modify cart in sticky footer modal
- Clear cart as needed
